# JSONFire

Simplifies work with network in iOS. Provides fundamental architecture for fast implementation of network requests.

## How To Get Started

- Copy content of `Source` folder to your project.

or

- Use `JSONFire` cocoapod

## Requirements

* iOS 9 and later
* Xcode 9 and later
* Swift 4
